# UPI Transactions Analytics

## Project Overview
This project analyzes UPI (Unified Payments Interface) transaction data using SQL, Python, and Power BI. The goal is to identify transaction patterns, payment trends, customer behavior, and generate interactive business insights through data visualization.

## Tools & Technologies
- SQL
- Python (Pandas, NumPy, Matplotlib)
- Power BI
- CSV Dataset

## Project Workflow
1. Import the UPI transaction dataset.
2. Clean and preprocess the data using Python.
3. Perform SQL queries for data analysis.
4. Create an interactive Power BI dashboard.
5. Generate insights from transaction trends.

## Key SQL Analysis
- Total number of transactions
- Total transaction amount
- Average transaction value
- Daily and monthly transaction trends
- Transaction status analysis
- Payment method analysis
- Top users by transaction amount

## Python Analysis
- Data cleaning
- Missing value handling
- Descriptive statistics
- Transaction trend visualization
- Payment method analysis
- Correlation analysis

## Power BI Dashboard
The dashboard includes:
- Total Transactions KPI
- Total Transaction Amount
- Average Transaction Value
- Transaction Status Distribution
- Monthly Transaction Trend
- Payment Method Analysis
- Interactive Filters (Slicers)
